INSEE IRIS Geolocalization

Summary

Find an IRIS code from longitude/latitude coordinates.
Source code at the Git cdbdx website.

Data

Find the raw data at the Open Data French gouvernment website.

Access

Examples:

  • Get some information from an IRIS code: URL/pyris/iris/0508
  • Retrieve the IRIS code from a coordinate: URL/pyris/search/?lon=-0.5695&lat=44.3586
You also can read and test the REST API.
